# Knowledge Graph Schema

The first KG package boundary reserves space for curated mitochondrial sources:

- MitoCarta 3.0 genes and proteins
- Reactome and KEGG pathways
- Gene Ontology mitochondrial hierarchy rooted at `GO:0005739`
- MITOMAP mtDNA variants
- HPO and MONDO disease/phenotype concepts
- ChEBI metabolites

Phase 3 will define node labels, edge types, uniqueness constraints, and import
scripts. Phase 6 will add paper-derived triples with provenance down to chunk ID.
